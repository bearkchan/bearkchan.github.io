DockerFile介绍

1. CMD和ENTRYPOINT

  • cmd命令指定的是默认的命令，是可以被覆盖掉的。

2. ARG和ENV的区别

ARG是构建的时候才起效果的作用，运行的时候就没有用了。

ENV是在整个容器运行期间都有效的。

arg参数的值可以在构建的时候临时修改，使用–build-arg

3. ONBUILD

如果有镜像是基于当前镜像创建的时候，才会起作用。本身自己镜像构建的时候是不起作用的。
